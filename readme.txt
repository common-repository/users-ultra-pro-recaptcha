This is a free add-on for <a href='https://wordpress.org/plugins/users-ultra/'>Users Ultra Pro 3.0 Plugin.</a>  


== Installation ==

1. Upload `users-ultra-pro-recaptcha` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
